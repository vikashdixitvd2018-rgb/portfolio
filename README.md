## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/vikashdixitvd2018-rgb/portfolio.git
```

### 2. Navigate into the project

```bash
cd portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
